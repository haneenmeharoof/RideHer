### Team Name: Echo

### Hosted Project Link
https://ride-her.vercel.app/

### Project Description
Our project is a taxi for women. Its a website. You can login in to the page using your details and can choose a driver after your location and destination. 
SOS option is available during ride progress. Live map is available


### The Problem statement
Women are not always safe while taking taxi, we aim to build a website which helps them. 

### The Solution
Our website offers rides to women and the service is provided by female drivers.
